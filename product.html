One API.
Countless Utilities

Pelm provides a level of abstraction on top of the APIs of utility companies. We handle the authentication and querying logic for various utilities, so you can focus on what matters most.

Start querying energy data in two easy steps

  1. Permission a user's data with /auth/

    2. We’ll redirect the user to their utility provider to grant Pelm acess to their data and permission to share that data with you. Once the user completes the auth flow, we redirect them to the redirectUrl provided by you and we share a OAuth token with you.

  2. Query data with /intervals/

    3. Make a request to our endpoint to get user data, and we’ll send you back a JSON response based on the parameters you specify (time range, usage point, etc).
